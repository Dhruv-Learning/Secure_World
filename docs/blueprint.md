# **App Name**: Vanguard Security

## Core Features:

- Professional Security Guard Display: Showcase professional security guard imagery.
- Dynamic Statistics Display: Display animated counters for trained security guards, happy clients, client satisfaction, and security support.
- Service Card Animation: Animate service cards with hover effects, clear iconography, and descriptions for security guard services, corporate security, and more.
- Client Profiling: Generative AI tool helps categorize leads based on the information entered in the request form, using the categorized information the agent will understand the most suitable approach to convert them into clients.

## Style Guidelines:

- Primary color: Deep navy blue (#243A73) for a trustworthy and authoritative feel.
- Background color: Dark gray (#2D2D2D) to enhance contrast and give a premium look.
- Accent color: Gold (#FFD700) to convey reliability and high-quality service.
- Font pairing: 'Inter' (sans-serif) for body text, and 'Space Grotesk' (sans-serif) for headlines; gives a modern, readable, and professional appearance.
- Use shield, lock, and CCTV icons to represent security effectively.
- Implement smooth fade-in and slide animations on the Home page.
- Ensure fully responsive design across mobile, tablet, and desktop devices.
